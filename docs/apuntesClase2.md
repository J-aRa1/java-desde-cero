## **Operadores Aritmeticos**
Son los aperadores que nos permiten hacer operaciones matematicas como son:
- "+" para sumar variables
- "-" ára restar variables 
- "*" para multiplicar variables 
- "/" para dividir variables, pero en la división hay que abrir un parentesis debido a que aquí podemos tener resultados con decimales,
sin embargo, si yo asigno dos variables enteras y las divido, su resultado tambien va a ser un entero sin decimales, pero su una de las 2 variables 
es de tipo decimal el resultado va a ser decimal. 
- "%" para el módulo de una división es decir el residuo de una división, por ejemplo:
16 % 4 = 0

a = 3; y luego hacemos a+= 1; el resultado de a es 4 ya que es lo mismo que poner a = a + 1, y es exactamente lo mismo con
